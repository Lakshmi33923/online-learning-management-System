Python Programming

What is python?

Python is a programming language that is used for many purposes, including software development, data analysis, and web applications. It is a general-purpose language that is easy to learn and can run on many different platforms.

Data Types

python has several built-in data types, including:

  • int
  • float
  • double
  • char

Variables

python, variables are used to store data values. To declare a variable, you need to specify the data type and a name for the variable.

        int age;
                float temperature;
                char letter;

Control Structures

python provides several control structures, including:

  • if/else
  • for
  • while
  • switch

Arithmetic Operators

Arithmetic operators are used to perform mathematical operations on operands.

      + (addition)
              - (subtraction)
              * (multiplication)
              / (division)
              % (modulus)

Assignment Operators

Assignment operators are used to assign values to variables.

      =  (simple assignment)
              += (add and assign)
              -= (subtract and assign)
              *= (multiply and assign)
              /= (divide and assign)
              %= (modulus and assign)

Relational Operators

Relational operators are used to compare two operands.

      == (equal to)
              != (not equal to)
              < (less than)
              <= (less than or equal to)
              > (greater than)
              >= (greater than or equal to)

Logical Operators

Logical operators are used to perform logical operations on operands.

      && (logical AND)
              || (logical OR)
              ! (logical NOT)

Bitwise Operators

Bitwise operators are used to perform bitwise operations on operands.

      & (bitwise AND)
              | (bitwise OR)
              ^ (bitwise XOR)
              ~ (bitwise NOT)
              << (left shift)
              >> (right shift)

Ternary Operator

The ternary operator is a shorthand way of writing an if-else statement.

      variable = (condition) ? value1 : value2;

Sizeof Operator

sizeOf operator is used to find the size or memory allocated to a varible,expression
